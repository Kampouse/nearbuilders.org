Edit social links on the builder profile.

- Add website, GitHub, X, Telegram, and LinkedIn fields to the builder profile editor. Each field is an icon-prefixed input where the user enters just their handle; the full URL is composed on save.
- Prefill the link fields from the builder's NEAR Social profile so existing links can be kept, edited, or cleared.
- On the profile page, hide social link badges whose value is empty.
